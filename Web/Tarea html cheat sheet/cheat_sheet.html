HTML Cheat Sheet

a

La etiqueta <a> define un hipervínculo, que se utiliza para enlazar de una página a otra.

Ejemplo de uso:

<a href="https://www.youtube.com/watch?v=XB11CTLKlR0&ab_channel=KlassicLoL">Video gracioso</a>

p

La etiqueta <p> define un párrafo.

Ejemplo de uso:

<p>Aquí escribo un bloque de texto lmao.</p>

h1, h2, ... , h6

Las etiquetas <h1> a <h6> definen encabezados de diferentes tamaños, siendo <h1> el más grande y <h6> el más pequeño.

Ejemplo de uso:

<h1>Primero</h1>
<h2>Segundo</h2>

div

La etiqueta <div> define una división o sección en un documento HTML.

Ejemplo de uso:

<div>El contenido va aquí</div>

span

La etiqueta <span> se utiliza para agrupar elementos en línea en un documento.

Ejemplo de uso:

<p>King Crimson - <span style="color:red">red</span>.</p>

strong

La etiqueta <strong> se utiliza para indicar texto con una importancia fuerte.

Ejemplo de uso:

<p>Esto es una palabra <strong>fuerte</strong>.</p>

i

La etiqueta <i> define una parte del texto con una voz o estado de ánimo alternativo.

Ejemplo de uso:

<p>Esto es una palabra <i>cursiva</i>.</p>

u

La etiqueta <u> se utiliza para subrayar texto.

Ejemplo de uso:

<p>Esto es una palabra <u>subrayada</u>.</p>

b

La etiqueta <b> se utiliza para especificar texto en negrita.

Ejemplo de uso:

<p>Esto es una palabra <b>en negrita</b>.</p>

strike

La etiqueta <strike> se utiliza para poner una línea a través del texto.

Ejemplo de uso:

<p>Rule #1 of <strike>fight club</strike>: don't talk about the fight club.</p>

small

La etiqueta <small> hace que el tamaño de la fuente del texto sea un tamaño más pequeño (por ejemplo, de small a x-small).

Ejemplo de uso:

<p>Esto es una palabra <small>más pequeña</small>.</p>

em

La etiqueta <em> se utiliza para enfatizar el texto.

Ejemplo de uso:

<p>Esto es una palabra <em>enfatizada</em>.</p>

header

La etiqueta <header> define un encabezado para un documento o una sección.

Ejemplo de uso:

<header>Encabezado del sitio web</header>

section

La etiqueta <section> define una sección en un documento.

Ejemplo de uso:

<section>Contenido principal del artículo</section>

article

La etiqueta <article> define un artículo independiente en un documento.

Ejemplo de uso:

<article>Últimas noticias del día</article>

aside

La etiqueta <aside> define contenido secundario en una página.

Ejemplo de uso:

<aside>Barra lateral con enlaces relacionados</aside>

footer

La etiqueta <footer> define un pie de página para un documento o una sección.

Ejemplo de uso:

<footer>Pie de página con información de contacto</footer>

nav

La etiqueta <nav> define una sección de navegación en un documento.

Ejemplo de uso:

<nav>Menú de navegación principal</nav>

img

La etiqueta <img> se utiliza para mostrar una imagen en una página web.

Ejemplo de uso:

<img src="imagen.png" alt="Logo de la empresa">

br

La etiqueta <br> se utiliza para insertar un salto de línea en el texto.

Ejemplo de uso:

Este es un texto.<br>Este es otro texto en una nueva línea.

hr

La etiqueta <hr> se utiliza para insertar una línea horizontal en el documento.

Ejemplo de uso:

<hr>

form

La etiqueta <form> se utiliza para crear un formulario de entrada de datos en una página web.

Ejemplo de uso:

<form action="procesar.php" method="post">Formulario de contacto</form>

table

La etiqueta <table> se utiliza para crear una tabla en una página web.

Ejemplo de uso:

<table>...</table>

ul

La etiqueta <ul> se utiliza para crear una lista desordenada en una página web.

Ejemplo de uso:

<ul><li>Elemento 1</li><li>Elemento 2</li></ul>

ol

La etiqueta <ol> se utiliza para crear una lista ordenada en una página web.

Ejemplo de uso:

<ol><li>Paso 1</li><li>Paso 2</li></ol>
